Pluto is a benchmark for evaluating extract method refactoring recommendation approaches. At present it has two versions. Pluto 2.0 is most recent version.

Research

My research interest involves automating identification of design flaws (commonly known as code smells) and suggesting software refactorings to mitigate such flaws. In particular I have been working on Long method code smell and its associated refactoring, Extract method.

Resources/Downloads

Interested readers may find following books useful:
Refactoring: Improving the Design of Existing Code by Martin Fowler
This book catalogues a list of 22 code smells and suggests the corresponding refactorings for mitigation.
Fundamentals of Software Architecture by Mark Richards and Neal Ford
A software architecture book written by software practioners, keeping the students and professionals in the view.
Elegant Objects by Yegor Bugayenko
A person with basics of object oriented programming could benefit from reading this title.
